# 任务：运行 Forkly 开发预览

## 目标

启动 Forkly 的开发预览环境，用于快速查看 Web 控制台界面，不需要先打包 DMG。

## 项目信息

- 项目名称：forkly
- 项目路径：`/Users/totti/个人/project/mod/forkly`
- Web 目录：`web`
- 本地预览地址：`http://127.0.0.1:5173/`
- 远程地址：`https://github.com/tottiqq7788/forkly.git`

## 启动方式

1. 进入项目根目录：

   ```bash
   cd /Users/totti/个人/project/mod/forkly
   ```

2. 安装依赖：

   ```bash
   cd web
   npm install
   ```

3. 启动 Vite 开发服务器：

   ```bash
   npm run dev -- --host 127.0.0.1
   ```

4. 在浏览器打开：

   ```text
   http://127.0.0.1:5173/
   ```

## 可选：连接真实本地 API

如果需要预览真实数据和完整会话流程，请在另一个终端启动 Go 服务：

```bash
cd /Users/totti/个人/project/mod/forkly
go run ./cmd/forkly
```

然后从菜单栏 Forkly 图标点击「打开控制台」。

## 注意事项

- 仅查看界面时，运行 `npm run dev` 即可。
- 若页面请求 `/local-api` 报错，说明没有启动 Go 本地 API。
- 不要用开发预览替代 macOS 打包验证。
